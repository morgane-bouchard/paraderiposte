Couleur Couleur Hex Codes

Color-hex fournit des informations sur les couleurs, y compris les modèles de couleurs (RVB, HSL, HSV et CMJN), les couleurs triadiques, les couleurs monochromes et les couleurs analogues calculées dans une page couleur. Color-hex.com génère également un code CSS simple pour la couleur sélectionnée. Des exemples d’éléments HTML sont également affichés sous la page de détail des couleurs. Tapez simplement le code de couleur à 6 chiffres dans la case ci-dessus et appuyez sur Entrée.
